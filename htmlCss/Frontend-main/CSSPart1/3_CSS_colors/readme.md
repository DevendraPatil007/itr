topic : color 
 we can color the font and background

 there are three ways we can give color 
 1. using direct color
  ex :- color : color_Name 
        color : red 
2. using hex color 
  ex color:  #hexCode 
     color : #fffffff
3. using rgb (255,0,100)

for Background 
background-color: aqua
